# 完整技术栈

用户完整技术栈：前端 Vue + JS + 二三维地图库（WebGIS：二维用 Mapbox GL JS 和 OpenLayers，三维用 CesiumJS），后端 FastAPI，数据库 PostGIS；后续计划学习 RAG/Agent。这意味着讲对象类型/interface 时可以适当用地理数据（坐标、图层、要素属性）作为例子帮助落地，但地图库本身的类型定义、FastAPI/PostGIS、RAG/Agent 相关 SDK 类型都明确不在本 mission 范围内（见 [[MISSION.md]] Out of scope）——先把 TS 语言本身学扎实，具体库的类型用法等用户进入项目阶段再开新方向。
