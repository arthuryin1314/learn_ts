TypeScript 基础类型标注

配合《为什么用 TypeScript + 基础类型标注》食用

变量

let age: number = 18;
let name: string = "Ada";
let isDone: boolean = false;

// 大多数时候可以省略标注,让 TS 自动推断:
let age2 = 18; // 推断为 number,效果一样

数组

let ids: number[] = [1, 2, 3];
let names: string[] = ["a", "b"];
let ids2: Array<number> = [1, 2, 3]; // 等价写法

函数

// 参数类型写在参数名后面,返回值类型写在 ) 后面
function add(a: number, b: number): number {
  return a + b;
}

// 箭头函数同理
const add2 = (a: number, b: number): number => a + b;

// 没有返回值时用 void
function log(msg: string): void {
  console.log(msg);
}

对象(内联类型)

function printUser(user: { name: string; age: number }): void {
  console.log(user.name, user.age);
}

特殊类型

any关闭该变量的类型检查,尽量少用
unknown比 any 安全的"未知类型",使用前必须先收窄
void函数没有返回值
never函数永远不会正常返回(比如一直抛异常)
记住
类型标注不会改变代码运行时的行为——TypeScript 编译后类型信息全部被擦除,产物就是普通 JS。类型检查只发生在写代码/编译的时候。
