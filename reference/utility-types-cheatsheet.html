工具类型(Utility Types)

配合《工具类型(Utility Types)》食用

Partial<T> / Required<T>

interface LayerConfig {
  name: string;
  visible: boolean;
  sourceUrl: string;
}

type Patch = Partial<LayerConfig>;  // 所有字段变可选
type Full  = Required<LayerConfig>; // 所有字段变必填

Pick<T, K> / Omit<T, K>

type Summary = Pick<LayerConfig, "name" | "visible">; // 只留下这几个字段
type NoSource = Omit<LayerConfig, "sourceUrl">;       // 排除这几个字段

Readonly<T>

type Frozen = Readonly<LayerConfig>; // 所有字段变 readonly

Record<K, V>

const visibility: Record<string, boolean> = { roads: true, lake: false };
// key 名字不固定,但每个 value 类型统一

组合

type LayerPatch = Partial<Pick<LayerConfig, "visible" | "sourceUrl">>;
// 先 Pick 出两个字段,再整体 Partial
记住
这些工具类型不产生新数据,只是"从已有类型的字段列表派生新的字段列表",本质上是编译期的类型运算。
