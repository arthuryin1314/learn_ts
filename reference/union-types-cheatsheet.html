联合类型与类型收窄

配合《联合类型与类型收窄》食用

联合类型 |

// 值只能是这几种类型之一
function formatFeatureId(id: string | number): string {
  // ...
}

type LayerType = "vector" | "raster"; // 字符串字面量联合类型

为什么需要收窄

interface VectorLayerConfig {
  type: "vector";
  sourceUrl: string;
}
interface RasterLayerConfig {
  type: "raster";
  tileSize: number;
}
type LayerConfig = VectorLayerConfig | RasterLayerConfig;

function loadLayer(config: LayerConfig) {
  console.log(config.sourceUrl);
  // 报错:Property 'sourceUrl' does not exist on type 'RasterLayerConfig'.
}

typeof 收窄(适合基础类型)

function formatFeatureId(id: string | number): string {
  if (typeof id === "number") {
    return `#${id}`; // 这里 TS 知道 id 是 number
  }
  return id.toUpperCase(); // 这里 TS 知道 id 是 string
}

可辨识联合收窄(适合对象)

function loadLayer(config: LayerConfig): void {
  if (config.type === "vector") {
    console.log(config.sourceUrl); // 安全,收窄成 VectorLayerConfig
  } else {
    console.log(config.tileSize); // 安全,收窄成 RasterLayerConfig
  }
}
记住
判断用的公共字段(比如 type)必须是字符串字面量类型("vector" 而不是 string),TS 才能靠它精确区分是哪一种——这种模式叫"可辨识联合"(discriminated union)。
