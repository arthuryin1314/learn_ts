泛型入门

配合《泛型入门》食用

泛型函数

function identity<T>(value: T): T {
  return value;
}

const y = identity("hello"); // T 被推断为 string

泛型约束(extends,表示"必须满足这个形状")

interface HasLength {
  length: number;
}

function logLength<T extends HasLength>(value: T): T {
  console.log(value.length);
  return value;
}

泛型接口

interface Feature<G> {
  type: "Feature";
  geometry: G;
  properties: Record<string, unknown>;
}

const pointFeature: Feature<{ type: "Point"; coordinates: [number, number] }> = {
  type: "Feature",
  geometry: { type: "Point", coordinates: [116.4, 39.9] },
  properties: {},
};

泛型 class

class LayerGroup<T extends MapLayer> {
  private layers: T[] = [];
  add(layer: T): void {
    this.layers.push(layer);
  }
}

const vectorGroup = new LayerGroup<VectorLayer>();
// vectorGroup.add() 现在只接受 VectorLayer
记住
泛型里的 extends 表示"约束/必须满足这个形状",和 class 继承的 extends(表示"继承自")不是同一件事,虽然写法一样。
